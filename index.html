# about
About Me

I'm Amit Manjarly, an ML engineer and researcher from Arizona, USA. My work focuses on training and fine-tuning large language models, building intelligent recommendation systems, and applying deep learning to real-world problems.

My latest project involved fine-tuning a 3B parameter Llama model on GSM8K using the Self-Taught Reasoner (STaR) methodology — demonstrating that clever data bootstrapping can outperform naive supervised fine-tuning.

I'm also a competitive programmer — actively solving algorithmic problems in Java and C++ on LeetCode and Codeforces, and sharing knowledge through my YouTube channel.

A personal AI agent that learns from your documents, remembers everything across sessions, and searches the web when it doesn't know something — all with per-user accounts and isolated memory. Feed it a PDF, URL, or plain text and it stores knowledge in a personal vector database.
